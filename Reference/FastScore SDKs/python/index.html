FastScore SDK for Python¶

Release 1.7

An example of an interaction with FastScore:

>>> import fastscore
>>> connect = fastscore.Connect("https://localhost:8000")
>>> model = fastscore.Model('model-1')
>>> model.source = '...'
>>> mm = connect.lookup('model-manage')
>>> model.update(mm)
>>> mm.models.names()
['model-1']
>>> del mm.models['model-1']
